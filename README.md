# Практическая работа по JavaScript и Git

## Описание
В проекте реализованы:
- функции JavaScript (работа с числами, строками, массивами)
- работа с объектами
- взаимодействие с DOM

## Используемые технологии
- HTML
- JavaScript
- Git
- GitHub Desktop

## Резервное копирование проекта
Резервное копирование выполнялось с помощью GitHub Desktop.

После внесения изменений в проект:
1. Все изменения отображались в программе.
2. Выполнялся Commit с описанием изменений.
3. Затем изменения отправлялись в удалённый репозиторий с помощью кнопки Push.

Таким образом, каждая версия проекта сохранялась на GitHub и могла быть восстановлена при необходимости.

## Файлы проекта
- index.html — структура страницы
- script.js — логика JavaScript

## Вывод
В ходе выполнения практической работы были изучены основы языка JavaScript, включая работу с функциями, строками, массивами и объектами. Также были освоены базовые принципы работы с системой контроля версий Git. Использование GitHub Desktop позволило удобно сохранять изменения и создавать резервные копии проекта. В результате была получена практика разработки и сохранения веб-проекта.
