Практическая работа №1
Тема: Использование Git и создание резервных копий проекта

Цель работы:
Изучить основы работы с системой контроля версий Git и научиться создавать резервные копии проекта.

Ход работы:
1. Установлен Git и проверена его работоспособность.
2. Выполнена настройка пользователя (имя и email).
3. Создан локальный репозиторий с помощью команды git init.
4. Добавлены файлы проекта (index.html, script.js, README.md).
5. Создан первый коммит.
6. Создан удалённый репозиторий на GitHub.
7. Выполнено подключение локального репозитория к удалённому.
8. Загружены файлы проекта в GitHub.
9. После изменений выполнялись команды git add, git commit и git push.

Вывод:
В ходе практической работы были освоены основные команды Git. Git позволяет отслеживать изменения в проекте, хранить историю версий и создавать резервные копии. Использование GitHub обеспечивает сохранность данных и удобство совместной работы.
